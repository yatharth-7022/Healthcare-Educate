1. User clicks subcategory
2. Navigate to: 👉 /dashboard/practice/[category]/[subcategory]/setup
3. Step 1 → Choose number of question sets
4. Step 2 → Confirm practice settings

🎯 Purpose
Let the user choose how many question sets they want.

🖼️ UI (MATCH REFERENCE IMAGE EXACTLY)

* Dark/light theme (same as dashboard)
* Clean vertical list of selectable options

🧠 Section Title
Untimed practice
Subtext
“Question numbers per set will vary by section and topic.”

📋 Options List (MANDATORY)
Display selectable rows:

* 4 question sets

* 7 question sets
* 8 question sets
* 9 question sets
* 10 question sets
⚠️ Maximum = 10 only

🎨 Row Design
Each row must include:
* Small icon on left (subtle)
* Text label (e.g. “4 question sets”)
* Full row clickable

🧩 Interaction
* Only one selection allowed
* Selected row:
    * Highlighted background
    * Slight elevation or glow
* Hover state:
    * Subtle highlight

📌 Behavior
* When user selects an option → store value in state
* Then show Continue / Next button
